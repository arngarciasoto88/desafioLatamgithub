Front end

El front-end se refiere a la parte visual de una aplicación web que los usuarios interactúan directamente. Incluye el diseño, la estructura y el comportamiento de la interfaz de usuario, utilizando tecnologías como HTML, CSS y JavaScript. El objetivo principal del front-end es ofrecer una experiencia de usuario atractiva y funcional.

Base de datos

Una base de datos es un sistema organizado para almacenar, gestionar y acceder a datos. Utiliza un sistema de gestión (DBMS) para permitir operaciones como creación, consulta, actualización y eliminación de datos. Puede ser relacional (tablas y SQL) o no relacional (NoSQL, para datos no estructurados). Su objetivo es facilitar la administración eficiente de la información.

Back End

El back-end gestiona la lógica del servidor, bases de datos y la comunicación entre el servidor y el cliente en una aplicación web. Utiliza lenguajes como Python, Java o Node.js y bases de datos para procesar y almacenar datos. Asegura que la aplicación funcione correctamente y maneje los datos de manera segura.

Lenguajes que he visto

Python

Python es un lenguaje de programación versátil y fácil de aprender, ideal para desarrollo web, ciencia de datos y automatización. Ofrece una sintaxis clara y una amplia biblioteca estándar. Es conocido por su comunidad activa y soporte extensivo.

PL/SQL

PL/SQL combina SQL con características de programación. Facilita la creación de aplicaciones para gestionar bases de datos Oracle. Permite la automatización de tareas complejas y la ejecución de procesos almacenados.

Java

Java es versátil y ampliamente utilizado para aplicaciones web, móviles y empresariales. Es conocido por su portabilidad y robustez. La plataforma Java permite desarrollar y ejecutar aplicaciones en cualquier entorno con una máquina virtual Java (JVM).
